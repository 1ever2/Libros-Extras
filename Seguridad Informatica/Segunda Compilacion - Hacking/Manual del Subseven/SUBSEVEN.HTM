Ip notify

Nos sirve para seleccionar las maneras de avisarnos cuando el server se conecta a la red.

Podemos hacerlo de 3 maneras diferentes.

Estas utilidades las tiene el editserver para crear el server, las tenemos en el cliente, para cambiarlas en cualquier momento.

Icq notify Esta es la primera y para mí la más cómoda de avisar de cuando el  server esta online, es por medio del icq, programa que te permite saber cuando se conectan tus amigos a tiempo real.

UIN: Aquí tenemos que poner el numero de icq nuestro para que pueda llegarnos el mensaje.

Victim: En este pondremos el nombre del mensaje para saber de donde viene.

Enable Es para activar ese de aviso.

Disable Lo contrario de enable, para desactivar el aviso del server.

IRC notify Este es uno de los mas usados por todo el mundo, la verdad que a mi particularmente es uno de los que no me hace gracia. El server enviara un mensaje a un canal del irc que tu quieras.

Server: Aquí pondremos la dirección del servidor de irc, al cual enviara el mensaje.

Port: Es el numero de puerto del server irc, que por defecto es 6667 y no hace falta cambiarlo.

Channel or nickname: En este campo pondremos el canal del irc que queremos para los mensajes, por ejemplo #sub7, también podemos poner el nickname de la persona, así recibes el mensaje en tu nick y es mas privado.

E-mail notify Este es el ultimo modo de aviso, el cual es por la cuenta de e-mail, pero es algo lento.

SMTP server: Aquí tienes que poner el servidor de correo entrante de tu cuenta de e-mail.

Bueno amigos con esto termino el articulo de connection, espero les ayude a entender este magnifico programa.
